# THOR 34-DOF Humanoid Optimization-Based Whole-Body Control Simulation

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

A high-fidelity, from-scratch implementation of **Contact-Implicit Model Predictive Control** with **Hierarchical Quadratic Programming Whole-Body Control** for the **THOR 34-DOF humanoid robot**.

> Work in progress.

## References

1. Hopkins, M.A. & Leonessa, A. (2015). *Int. J. Humanoid Robotics*, 12(3).
2. Orin, D.E. et al. (2013). *Autonomous Robots*, 35(2-3), 161-176.
3. Featherstone, R. (2008). *Rigid Body Dynamics Algorithms*. Springer.
4. Escande, A. et al. (2014). *IJRR*, 33(7), 1006-1028.
5. Meduri, A. et al. (2023). *IEEE TRO*, 39(2), 905-922.
